Datoru aizsardzība

  • Izmantojiet ļaunprogrammatūras novēršanas programmu — programmas ļaunprogrammatūras instalēšana un tās atjaunināšana var palīdzēt datoram sevi noturēt pret vīrusiem un citu ļaunprogrammatūru (ļaunprogrammatūru). Microsoft Defender ir bezmaksas ļaunprogrammatūras novēršanas programmatūra, kas iekļauta programmā Windows, un tā tiek automātiski atjaunināta, izmantojot Windows Update. Pastāv arī ļaunprogrammatūras novēršanas produkti, ko veido citi uzņēmumi, no kuriem varat izvēlēties.
  • Neatveriet e-pasta ziņojumus no nepazīstamiem sūtītājiem vai neatpazīto e-pasta pielikumus — Daudzi vīrusi ir pievienoti e-pasta ziņojumiem un tiks izplatīti, tiklīdz atverat pielikumu. Ieteicams neatvērt negaidīti atsūtītus pielikumus. Papildinformāciju skatiet rakstā Aizsardzība pret pikšķerēšanu.
  • Izmantojiet uznirstošo logu bloķētāju ar interneta pārlūkprogrammu — uznirstošie logi ir mazi pārlūkprogrammas logi, kas tiek rādīti virs skatītās tīmekļa vietnes. Lai gan tos parasti veido reklāmdevēji, tajos var būt iekļauts ļaunprātīgs vai nedrošs kods. Uznirstošo elementu bloķētājs var neļaut dažu vai visu šo logu parādīšanu. Uznirstošo logu bloķētājs lapā Microsoft Edge ieslēgts pēc noklusējuma.
  • Ja izmantojat programmu Microsoft Edge, pārliecinieties, vai līdzeklis SmartScreen ir ieslēgts — līdzeklis SmartScreen programmā Microsoft Edge palīdz aizsargāties pret pikšķerēšanas un ļaunprogrammatūras uzbrukumiem, brīdina, ja par tīmekļa vietni vai lejupielādes atrašanās vietu tiek ziņots kā nedroša. Papildinformāciju skatiet rakstā Kā izmantot smartScreen, lai mani aizsargātu Microsoft Edge?
  • Pievērsiet uzmanību Windows SmartScreen paziņojumiem — esiet uzmanīgs, pirms palaižot neatpazītās programmas, kas lejupielādētas no interneta. Nezināmas lietojumprogrammas biežāk izrādās nedrošas. Kad lejupielādējat un palaižat lietojumprogrammu no interneta, SmartScreen izmanto informāciju par lietojumprogrammas reputāciju, lai brīdinātu jūs, ja šī lietojumprogramma nav labi zināma un var būt ļaunprātīga.
  • Regulāri Windows — periodiski korporācija Microsoft izlaiž īpašus drošības atjauninājumus, kas var palīdzēt aizsargāt jūsu datoru. Šie atjauninājumi palīdz izvairīties no vīrusu un citas ļaunprogrammatūras uzbrukumiem, izlabojot iespējamos drošības caurumus.
  • Izmantojiet interneta pārlūkprogrammas konfidencialitātes iestatījumus — dažas tīmekļa vietnes var mēģināt izmantot jūsu personisko informāciju, lai reklamētu, krāpniecisku darbību un identitātes zādzību. Visām modernajās pārlūkprogrammās ir konfidencialitātes iestatījumi, ko varat iespējot, lai kontrolētu, kādas vietnes var redzēt vai darīt.
